# Proyek-Akhir-Kecerdasan-Artifisial---Deteksi-Pelecehan-dari-Data-Teks
Proyek ini mengimplementasikan algoritma Naive Bayes dan Random Forest untuk mendeteksi pelecehan dalam teks. Menggunakan teknik Natural Language Processing (NLP) seperti TF-IDF dan Word2Vec untuk ekstraksi fitur, serta evaluasi model dengan akurasi, precision, recall, dan F1-score.
## Fitur Utama:
- **Klasifikasi Teks**: Menggunakan **Naive Bayes** dan **Random Forest** untuk mengklasifikasikan teks.
- **Teknik NLP**: Implementasi **TF-IDF** dan **Word2Vec** untuk ekstraksi fitur dari teks.
- **Evaluasi Model**: Penggunaan metrik **akurasi**, **precision**, **recall**, dan **F1-score** untuk mengukur kinerja model.
- **Aplikasi Web**: Dibangun menggunakan **Streamlit** untuk memungkinkan pengguna mendeteksi pelecehan teks secara real-time.
## Teknologi yang Digunakan:
- **Python**
- **Naive Bayes**
- **Random Forest**
- **Natural Language Processing (NLP)**
- **TF-IDF**, **Word2Vec**
- **Streamlit**
